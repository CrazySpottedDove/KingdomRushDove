# LÖVE2D Graphics API 开销参考表

| API调用 | 估计开销(μs) | 级别 | 说明 |
|---------|-------------|------|------|
| `G.getColor()` | 0.1-0.3 | 极低 | 简单的状态读取 |
| `G.push()` / `G.pop()` | 0.2-0.5 | 极低 | 栈操作，但频繁调用累积 |
| `G.setColor(r,g,b,a)` | 0.3-0.8 | 低 | 设置颜色状态 |
| `G.translate(x,y)` | 0.5-1.0 | 低 | 矩阵运算 |
| `G.scale(x,y)` | 0.5-1.0 | 低 | 矩阵运算 |
| `G.rotate(r)` | 0.5-1.0 | 低 | 矩阵运算 |
| `G.draw(image, x, y)` | 2-10 | 中 | 取决于图像大小和批处理 |
| `G.rectangle("fill", ...)` | 3-8 | 中 | 简单几何体绘制 |
| `G.setFont(font)` | 10-30 | 高 | 字体状态切换 |
| `G.printf(text, ...)` | 20-100 | 高 | 文本排版+渲染，取决于文本长度 |
| `G.setCanvas(canvas)` | 15-50 | 高 | 渲染目标切换 |
| `G.clear()` | 10-30 | 高 | 清空缓冲区 |
| `G.stencil(fn)` | 50-200 | 极高 | 模板缓冲操作+函数调用 |
| `G.setStencilTest()` | 20-50 | 高 | 模板测试状态切换 |
| `G.setShader(shader)` | 30-100 | 高 | 着色器切换 |
